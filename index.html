Intro

Hello there! I'm a Civil Engineering graduate who's ventured into the dynamic and ever-evolving world of Machine Learning and AI. My journey here wasn't a straight path, though. I was once on the cusp of diving into a PhD in Civil Engineering, but a burning question changed my course: "What's an exciting and future-proof career where I can harness cutting-edge technologies to build a futuristic world?" AI and advanced machine learning were the answers that resonated with me. While I was managing a lab and dabbling in machine learning, something just clicked. This was what I wanted my future to look like. So, I made a bold move. Instead of pursuing that PhD, I decided to leap into a Master's in Computer Science. Projects

Beginner Level Projects

Simple Linear Regression

I've developed a model that predicts vehicle CO2 emissions from various vehicle characteristics. My project not only applies fundamental regression techniques but also rigorously ensures that the data adheres to the assumptions necessary for linear regression validity. This analysis serves as a practical intersection of environmental concern and data science acumen. Click here for Simple Linear Regression GitHub Repo.

Multiple Linear Regression

In my multiple linear regression project, I integrated various vehicle characteristics to assess their combined impact on CO2 emissions. By mastering the nuances of multivariate analysis, I ensured the integrity of the data and the model's assumptions, leading to a more comprehensive understanding of emission factors. Click here for Multiple Linear Regression(1) GitHub Repo, Multiple Linear Regression(2) GitHub Repo.

Decision Tree & Regression Trees

In my decision tree project, I created a model to determine the appropriate medication for patients based on their unique characteristics. This involved transforming categorical variables with label encoding and constructing the decision tree using the entropy method. Additionally, I expanded my expertise to regression trees, utilizing scikit-learn's Decision Tree Regressor library to predict housing prices, honing my skills in both classification and regression tasks within decision tree frameworks. Click here for Decision Tree GitHub Repo.

Logistic Regression

In my logistic regression project, I analyzed customer data to predict churn, employing various solver methods like 'liblinear' and 'lbfgs' to optimize the performance of the sklearn LogisticRegression model. By evaluating the model's accuracy through precision and recall metrics, I gained insights into customer retention and the nuances of predictive modeling Click here for Logistic Regression GitHub Repo.

Support Vector Machine (SVM)

In my Support Vector Machine (SVM) project, I explored the classification of cells as malignant or benign, aiming to identify potential cancer. I utilized SVM's robust capabilities to analyze cell features and applied the Jaccard score to measure the accuracy of the model's predictions, providing valuable insights into the effectiveness of SVM in medical diagnostics. Click here for SVM GitHub Repo.
